# Determinantal Ideals in Lean 4

This repository contains a Lean 4 / Mathlib formalization project on generic minors and determinantal ideals.

## Project description

The main goal of this project is to formalize basic material around the determinantal ideals of a generic matrix, with a longer-term goal of formalizing Gröbner basis results for minors under suitable diagonal term orders.

At the current stage, the project includes:

- the generic `m × n` matrix over `MvPolynomial (Fin m × Fin n) k`
- index data for `t × t` minors
- the minors themselves
- the determinantal ideal generated by all `t × t` minors
- finiteness and finite-generation results for the family of minors
- evaluation lemmas from the generic matrix to a concrete matrix
- sign and permutation lemmas for determinants
- infrastructure toward diagonal term orders and leading-term statements for minors

## Current file structure

- `MyProject/Determinantalideals/basic.lean`  
  Basic definitions: generic matrix, minor indices, minors, minor sets.

- `MyProject/Determinantalideals/Finite.lean`  
  Finiteness results for `MinorIndex` and the set of minors.

- `MyProject/Determinantalideals/Eval.lean`  
  Evaluation of generic minors at concrete matrices.

- `MyProject/Determinantalideals/Ideal.lean`  
  Determinantal ideals and finite generation.

- `MyProject/Determinantalideals/Sign.lean`  
  Sign lemmas for permutations and determinant bridges.

## Longer-term goal

The long-term objective is to formalize the classical theorem that the `t × t` minors of a generic matrix form a Gröbner basis under a suitable diagonal term order.

## Status

This repository is currently a research/formalization project rather than an upstreamed Mathlib contribution. The code is being gradually reorganized toward Mathlib style, with attention to:

- smaller imports
- better documentation
- cleaner API design
- possible future upstreamable components
