# Determinantal Ideals in Lean 4

This repository contains a Lean 4 / Mathlib formalization project on generic minors and determinantal ideals.

## Project description

The project studies determinantal ideals of a generic matrix over the polynomial ring

`MvPolynomial (Fin m × Fin n) k`.

The current formalization includes foundational definitions and lemmas for:

- the generic `m × n` matrix,
- index data for `t × t` minors,
- the minors themselves,
- the determinantal ideal generated by all `t × t` minors,
- finiteness and finite-generation results,
- evaluation at concrete matrices,
- compatibility under coefficient ring homomorphisms,
- sign and permutation lemmas for determinant expansions,
- exponent vectors and permutation terms of minors,
- initial infrastructure for diagonal term orders and leading-term statements.

## Long-term goal

The long-term goal is to formalize the classical theorem that the `t × t` minors of a generic matrix form a Gröbner basis under a suitable diagonal term order.

## Current status

This repository is currently a research/formalization project rather than an upstreamed Mathlib contribution.

Some foundational files have been reorganized with Mathlib style in mind, including documentation, import cleanup, and API organization. More advanced files are still being refined and may continue to change.
